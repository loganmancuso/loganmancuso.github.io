# custom_terminal

` git clone git@gitlab.com:logmancuso/custom_terminal.git `

add this to the end of ~/.bashrc in the $HOME directory
```
for f in $HOME/.custom_terminal/*; do
   if [ ${f: -3} == ".sh" ]
   then
        . $f
   fi
done
```

then run exec bash
